- **🔧 ATS track** — companies with a public ATS API ([`hardware_companies.yml`](hardware_companies.yml)), scraped directly. Precise, with real apply links.
- **🔗 LinkedIn track** — companies without a scrapeable ATS ([`linkedin_companies.yml`](linkedin_companies.yml)) — Apple, Google, AMD, the big semis/EDA, and stealth AI-silicon startups — sourced via LinkedIn (Apify).
