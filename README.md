# My Hardware / ASIC / DV / RTL Job Tracker

A personal, auto-updating watchlist of entry-level **hardware engineering, ASIC/SoC design, Design Verification (DV), RTL/digital design, FPGA, and physical design** roles — summer internships, fall/spring co-ops, and new-grad / early-career full-time positions realistically gettable with **< 3 years of experience**, in the **US & Canada**.

Two automated tracks, both filtered by the same classifier ([`hw_classify.py`](.github/scripts/hw_classify.py)):

- **🔧 ATS track** — companies with a public ATS API ([`hardware_companies.yml`](hardware_companies.yml)), scraped directly. Precise, with real apply links.
- **🔗 LinkedIn track** — companies without a scrapeable ATS ([`linkedin_companies.yml`](linkedin_companies.yml)) — Apple, Google, Amazon, AMD, the big semis/EDA, and stealth AI-silicon startups — sourced via LinkedIn (Apify).
